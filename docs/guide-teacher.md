# Guide d'utilisation - TEACHER

## Objectif du role
L'enseignant gere la presence, les notes et le suivi des classes assignees.

## 1) Mes classes
- Consulter la liste des classes et des eleves.

## 2) Presences
- Selectionner la classe et la date.
- Marquer Present/Absent.
- Enregistrer le pointage.

## 3) Notes
- Selectionner classe, matiere, type d'evaluation.
- Saisir les notes et appreciations.
- Enregistrer et verifier les moyennes.

## 4) Emploi du temps
- Consulter votre planning hebdomadaire.

## 5) Messages
- Communiquer avec les parents si disponible.

## Bonnes pratiques
- Saisir les presences le jour meme.
- Verifier les notes avant validation.

## Depannage
- Classe introuvable : contacter l'Admin/RH pour l'affectation.
