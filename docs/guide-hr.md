# Guide d'utilisation - HR

## Objectif du role
Le RH gere le personnel, les pointages et les conges.

## 1) Personnel
- Consulter les fiches employes.
- Rechercher par nom ou role.
- Mettre a jour les informations si autorise.

## 2) Pointages
- Consulter les pointages du personnel.
- Filtrer par date, employe, statut.
- Supprimer un pointage en cas d'erreur.

## 3) Conges
- Enregistrer une demande de conge.
- Approuver ou rejeter.
- Suivre l'historique des conges.

## 4) Rapports
- Exporter les rapports RH (PDF/Excel selon ecran).

## Bonnes pratiques
- Verifier les statuts (present, retard, conge).
- Tenir a jour les dossiers RH.

## Depannage
- Employe manquant : verifier le compte utilisateur et le role.
